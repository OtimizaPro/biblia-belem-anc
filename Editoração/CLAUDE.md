## Regras Críticas (Resumo)

| Regra                   | Descrição                                         |
| ----------------------- | ------------------------------------------------- |
| **yhwh**                | Sempre minúsculo — NUNCA "YHWH"                   |
| **"livrinho"**          | Minúsculo = esta obra                             |
| **"Livrinho"**          | Maiúsculo = livro bíblico (Desvelação 10)         |
| **Designações divinas** | Grafia original + transliteração — NUNCA traduzir |
| **"Desvelação"**        | Termo correto — NUNCA "Apocalipse"                |

### Icebox — Material em Espera

Cada capítulo pode conter uma subpasta `Icebox/`. **Icebox NÃO faz parte do manuscrito oficial.**

| Regra  | Descrição                                                                                                              |
| ------ | ---------------------------------------------------------------------------------------------------------------------- |
| **I1** | Conteúdo em `Icebox/` é rascunho, material em espera ou descartado                                                     |
| **I2** | Icebox NUNCA deve ser citado como parte publicada do livrinho                                                          |
| **I3** | Para contar como parte do livrinho, o conteúdo deve estar em um dos 15 arquivos canônicos (ver VARREDURA_EDITORIAL.md) |
| **I4** | Icebox pode ser promovido a canônico apenas por decisão explícita do autor                                             |
